Brew Bubbles Settings

These settings apply to the Brew Bubbles as a whole

BrewPi Remix Target Settings

These settings control how Brew Bubbles talks to BrewPi Remix. To disable pushing to BrewPi Remix, delete the BrewPi Remix URL.

Fermentrack Target Settings

These settings control how Brew Bubbles talks to Fermentrack. To disable pushing to Fermentrack, delete the Fermentrack URL.

Google Sheets Settings

These settings control how Brew Bubbles talks to Google Sheets. To understand how to set up this feature (as well as a better explanation of these options), please read the Brew Bubbles documentation here.

Brewer's Friend Settings

These settings control how Brew Bubbles talks to Brewer's Friend. With a premium account, Brewer's Friend supports logging from gravity sensors such as the Tilt Hydrometer. To find your API key, log into your Brewer's Friend account and go to Profile > Account. Your API key will be a string of letters and numbers (not a URL).
